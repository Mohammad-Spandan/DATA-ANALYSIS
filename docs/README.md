# Documentation

## Project Documentation

This directory contains comprehensive documentation for the Agricultural Crop Data Analysis project.

## Files

### 1. DATA_DICTIONARY.md

Detailed documentation of all data fields, columns, and data structure.

**Contains**:
- Column descriptions
- Data types and formats
- Crop categories and IDs
- Production units
- Data quality notes
- Derived metrics definitions

### 2. ANALYSIS_APPROACH.md

Detailed explanation of the analysis methodology and framework.

**Contains**:
- Analysis phases and objectives
- Statistical methods
- Key metrics definitions
- Data quality considerations
- Limitations and assumptions
- Recommendations for improvements

### 3. API_REFERENCE.md

*(Optional - for when API is implemented)*

Documentation of available functions and API endpoints.

3. Document new features
